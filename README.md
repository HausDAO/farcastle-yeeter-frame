---

## Getting Started

```bash
git clone https://github.com/HausDAO/menagerie.git
```

Use node version

```bash
nvm use
```

```bash
bun install
```

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
